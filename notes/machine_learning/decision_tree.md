# Decision Tree

The **decision tree** model can be used for both classification and regression
tasks; howhever its structure suggests a propensity for classification.

La loro rappresentazione intrinseca di alberi li rende anche facilmente
interpretabili, fornendo infatti i punti critici in cui la loro classificazione
cambia. Sono infatti molto usati anche nell'ambito dell'Explainable AI.
